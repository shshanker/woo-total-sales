== Description ==

Woo Total Sales – This plugin will fetch total sales of specific woocommerce product, and then show into shop-archive, single product, backend single product metabox, product listing. 

Further more, Its has its own dashborad to overview and backend options to the woocommerce products setting to edit, to control and update "total sales" texts ( singular and plural ), "bar-chart" color and that texts too.

Contribution link: https://github.com/shshanker/woo-total-sales


### The Plugin Features:

* New feature - Visible only on single product page (frontend).

* New feature - Visible only on backend.

* Total sales dashboard under WooCommerce product setting section.

* Overview total sales, number of products, shipping costs and applied disocunt.

* Enable disable On-hold-order to Total Sales count.

* Backend Total Sales overview metabox for indivdual product.

* Total sales text - Enables WooCommerce shop manager to display total sales with product price with singular and plural forms of the counting number.

* Icon - For some sort of UI/UX aspect, it would automatically add the Bar-Chart icon infront of total sales text.

* Zero Sales - This funtionality has been disabled for the 0-sold products and would be automaticallay added if the taotal sales of the specific product exceed and equals to 1.

* Admin Setting - The plugin has facility to edit and update the "Total Sales" texts under woocommerce product general setting section. 

* Colourful icon and texts - And lastly you can set distinct bar-icon color and texts color if you really want to highilght or match the current theme UI/UX with total sales portion, other wise display as it is. 



== Installation ==
* Download to your plugin directory or simply install via WordPress admin interface.
* Activate.
* Go to specific woocommerce product section under setting menu.
* Go to general products setting section at where you can find 4 extra fields with "Total Sales Setting" titled.
* Set your texts and set color as described.
* Save your changes.
* Enjoy the plugin.
